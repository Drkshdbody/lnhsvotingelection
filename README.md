# Student Election System

## Setup:
1. Place all files in `xampp/htdocs/schoolelection/`
2. Make sure `election.db` is in the same folder
3. Start Apache in XAMPP
4. Visit: http://localhost/schoolelection

## Admin Login:
- Go to: http://localhost/schoolelection/admin_login.php
- Password: admin123

## Notes:
- Each student uses a unique login token.
- After voting, the token becomes invalid.
- Results update in real-time for admin.
